# YTMate - YouTube Downloader

YTMate is a powerful and easy-to-use Python web application that allows users to download both audio and video content from YouTube. Built with Flask and Pytube, this application offers a streamlined interface for fetching video information, choosing quality options, and downloading content for offline use.

## Features

- **Audio and Video Download:** Download both audio and video content from YouTube.
- **Quality Options:** Choose from a variety of quality options for downloaded files.
- **Responsive Design:** Enjoy a seamless experience on both desktop and mobile devices.
- **Real-Time Information:** Retrieve real-time video information to make informed download choices.

## Technologies Used

- Python
- Flask
- Pytube

## Getting Started

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/vishal-ravi/YTMate.git
   ```

2. **Install Dependencies:**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the Application:**
   ```bash
   python app.py
   ```

4. **Access the Application:**
   Open your web browser and navigate to [http://localhost:5000](http://localhost:5000)

## Usage

1. Enter the YouTube video URL in the provided input field.
2. Click on "Get Video Info" to fetch information about the video.
3. Explore available quality options for audio and video.
4. Select desired quality and click "Download" to save the content.
